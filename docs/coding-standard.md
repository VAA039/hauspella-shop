# HAUSPELLA Coding Standard

Версия: 1.0

---

## Назначение

Документ описывает единые правила разработки платформы HAUSPELLA.

Цель документа — обеспечить единый стиль кода, архитектуры и организации проекта независимо от количества разработчиков.

## Правило №1. Логическая структура моделей

Каждая модель должна состоять из логических блоков.

Используются только те блоки, которые действительно необходимы конкретной модели.

Возможные блоки:

- Identity
- Commerce
- Content
- SEO
- Marketing
- Media
- Classification
- Service
